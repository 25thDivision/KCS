# How to install PyTorch
---
First, open command window using **cmd** command

Type **nvidia-smi** on the window

Find your recommended CUDA version

Open any browser and go to https://pytorch.org/get-started/locally/
Or just click [https://pytorch.org/get-started/locally/|here]

Remember to check Stable, Pip, Python, and your recommended CUDA version.

Run the command below!
